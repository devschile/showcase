# Contribuir a devsChile ShowCase

Gracias por querer contribuir. A continuación se indican las formas recomendadas para añadir o actualizar una ficha de proyecto.

## Opciones para contribuir

- Edita `index.md` y añade tu proyecto siguiendo la plantilla de la sección "Plantilla de proyecto".
 - O crea un archivo nuevo dentro de `_projects/` llamado `nombre-del-proyecto.md` y coloca allí la ficha en formato Markdown (recomendado: incluye `title`, `author`, `repo`, `type`, `tech`, `description`).

Nota: El campo `permalink` es opcional. Si no lo incluyes, el sitio generará automáticamente la URL como `/projects/<slug>/` usando el `title` para crear el slug.

## Plantilla mínima de PR

1. Crea una rama desde `main` llamada `add/nombre-proyecto`.
2. Añade o edita el contenido según la plantilla.
3. Incluye enlace al repositorio y licencia (si aplica).
4. Abre un Pull Request con una descripción clara.

## Revisión

Los mantenedores revisarán la PR y podrán pedir cambios. Si tu proyecto es propietario, indica claramente qué información es pública.

## Contacto

Para dudas, escribe en el canal de Slack `#showcase` o abre un issue en este repositorio.
